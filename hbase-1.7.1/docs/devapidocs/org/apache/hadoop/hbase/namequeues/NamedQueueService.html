Interface NamedQueueService

  • All Known Implementing Classes:
    BalancerDecisionQueueService, SlowLogQueueService

    @InterfaceAudience.Private
public interface NamedQueueService
    In-memory Queue service provider for multiple use-cases. Implementers should be registered in LogEventHandler
